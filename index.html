HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • h1 represents headers, h1 is always going to be larger than the rest
  • div is used to create a division in a page
  • span creates an inline division
  • ul = unordered list
  • ol = ordered list
  • li = list items
  • img = image via web
  • src = location of image
  • br = break or space in between two blocks of content

CCS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • you can customize a color used by selecting its "hex" # number
  • css sheets are used to customize the look and feel of the site
  • Seperations of Concerns - allowing code to have its own responsibility
  • using a * while working on a css sheet, includes this ruling for ALL elements on the page

Git

  • git checkout -b branch name: creates a new branch and switches to it
